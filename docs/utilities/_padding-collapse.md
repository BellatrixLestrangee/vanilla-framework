---
collection: utilities
title: Padding collapse
---

Remove one or more paddings on an element.

### .u-no-padding

```html
<div class="u-no-padding">
    This div has no padding
</div>
```
<div class="theme__outline">
    <div class="theme__outline--inner u-no-padding">
        This div has no padding
    </div>
</div>
