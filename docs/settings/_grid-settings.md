title: Grid
---

Vanilla's default grid has 12 columns and 20px gutters. On large screens, each column takes up 6.3053% of the total width.

Setting  | Default value
 ------------- | -------------
`$grid-gutter-widths`   | 20px
`$grid-columns`   | 12
`$grid-max-width`   | 1030px
