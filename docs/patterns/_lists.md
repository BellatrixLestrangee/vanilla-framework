## .list

A basic list with no bullets, dividers or other decoration.

```html
<ul class="list">
    <li>Lorem</li>
    <li>Ipsum</li>
</ul>
```

<ul class="list">
    <li>Lorem</li>
    <li>Ipsum</li>
</ul>

## .list-state
A list with an icon bullet. Default is a tick.

```html
<ul class="list-state">
    <li>Lorem</li>
    <li>Ipsum</li>
</ul>
```

<ul class="list-state">
    <li>Lorem</li>
    <li>Ipsum</li>
</ul>
